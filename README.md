# TestProject
Projet de test de developpement JAVA

- Objectifs du projet détaillé : 
	1. Création du jeu du pendu en mode console
	2. Mise à jour du jeu avec : 
		- Utilisation d'un fichier pour générer le mot à trouver
		- Ecrire dans un fichier les statistiques de la partie : 
			- Nombre de tentatives
			- Nombre de mots trouvés
			- Temps de jeu 
	3. Lier l'application à une base de données
	4. Création d'un programme batch pour alimenter la base de données
